# The Last Train - Unity 2D Game 

**The Last Train** is a narrative-driven psychological puzzle game developed as an academic project.  
The game emphasizes moral choice, symbolic encounters, and story-focused gameplay.

---

## Overview
- **Genre:** Narrative-driven Puzzle Game  
- **Platform:** Windows, Android  
- **Graphics:** 2D Side-scrolling  
- **Focus:** Story-driven gameplay and player choices  

---

## Core Gameplay
Players explore surreal environments, solve logic-based puzzles, and face symbolic boss encounters.  
Player decisions influence narrative progression and determine multiple endings.
